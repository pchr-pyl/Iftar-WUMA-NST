📂 โครงสร้างไดเรกทอรี:
├── dist/            ⭐ นี่คือการสร้างสำหรับการผลิต (แนะนำให้ใช้โดยตรง)
│   ├── index.html   → เปิดไฟล์นี้ในเบราว์เซอร์เพื่อดูตัวอย่างเว็บไซต์
│   ├── assets/      → ไฟล์ CSS, JS และทรัพยากรอื่นๆ ที่ถูกรวม
│   └── ...
├── src/             📝 ไดเรกทอรีโค้ดต้นฉบับ (สำหรับการพัฒนาและแก้ไข)
│   ├── components/  → ส่วนประกอบ React
│   ├── App.jsx      → ส่วนประกอบแอปพลิเคชันหลัก
│   ├── index.js     → ไฟล์เริ่มต้น
│   └── ...
├── package.json     📦 การกำหนดค่าขึ้นอยู่กับโครงการ
├── README.md        📖 เอกสารประกอบโครงการ
└── ...

🚀 เริ่มต้นอย่างรวดเร็ว (สองตัวเลือก):
ตัวเลือกที่ 1: ใช้โดยตรง (แนะนำสำหรับผู้เริ่มต้น)
1. ไปที่ไดเรกทอรี dist/
2. ดับเบิลคลิกไฟล์ index.html
3. เว็บไซต์จะเปิดในเบราว์เซอร์ของคุณ!

ตัวเลือกที่ 2: โหมดพัฒนา (สำหรับผู้ใช้ที่มีประสบการณ์การเขียนโค้ด)
1. ตรวจสอบให้แน่ใจว่าได้ติดตั้ง Node.js แล้ว (เวอร์ชันที่แนะนำ: 16+)
2. เปิดเทอร์มินัล/บรรทัดคำสั่งในไดเรกทอรีรากของโครงการ
3. รัน: npm install
4. รัน: npm start
5. เซิร์ฟเวอร์พัฒนาจะเปิดโดยอัตโนมัติในเบราว์เซอร์ของคุณ

💡 เคล็ดลับ:
- ไดเรกทอรี dist/ มีการสร้างที่ปรับแต่งแล้วและสามารถนำไปใช้บนเซิร์ฟเวอร์ได้โดยตรง
- index.html ที่ระดับรากไม่ใช่เวอร์ชันสุดท้าย — ใช้ไฟล์ที่อยู่ใน dist/ เสมอ
- หากต้องการแก้ไขโค้ด ให้แก้ไขไฟล์ใน src/ และสร้างโครงการใหม่
- โครงการนี้ใช้เทคโนโลยี TypeScript และ Tailwind CSS
